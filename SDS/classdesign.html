4.3 Details of each class

User:

it contains all types of users. Such as: Participants and organizers. The attributes of this class is: Id; name; current institution.

Organizer Team:

this class is made up by one or multiple users. Any user can create a organizer team, add members. The attributes of this class is: Id; userId(foreign key); OrganizerName.

Competition:

Organizer team creates the competition and Participants can participate in the competition. There will also be a forum and discussion panel in every competition. The attributes of this class is: Id; userId(foreign key); name; venue; date; deadline.

Query:

Every competition have a forum. Where any one can ask about any query. And admin will answer to it. The attributes of this class is: Id; competitionId(foreign key).

Update:

Every competition will have a discussion page. Where the admin will post any update related with the Competition. Anyone will be able to comment on it. And admin will answer to it. The attributes of this class is: Id; competitionId(foreign key).

Category:

It is an abstract class. Every competition must belong to any category. The attributes of this class is: Id; competitionId(foreign key).
