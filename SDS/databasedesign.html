5.3 Details of each table

Table name: User
Attributes:
  • id - int
  • username - string
  • email - string
  • preferences - string

Table name: Organizer
Attributes:
  • user_id - int (Forign Key - User Table)
  • Team name - string

Table name: Competition
Attributes:
  • id - int
  • name - string
  • deadline - datetime
  • venue - string
  • description - string

Table name: Query
Attributes:
  • id - int
  • compettion_id - int (Forign Key - Competition Table)
  • user_id - int (Forign Key - User Table)
  • details - string
  • response - string

Table name: Updates
Attributes:
  • post-id - int
  • compettion_id - int (Forign Key - Competition Table)
  • post-details - string

Table name: Particited competitions
Attributes:
  • user_id - int (Forign Key - User Table)
  • post-id - int (Forign Key - Updates Table)
  • compettion_id - int (Forign Key - Competition Table)
  • isCompleted - boolean

Table name: Catagories
Attributes:
  • id - int
  • name - string
